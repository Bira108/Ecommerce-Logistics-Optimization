# Ecommerce-Logistics-Optimization
Business Intelligence and Logistic Analysis of shipping costs nationwide. How can we improve business profit based on the shipping of multiple items?
README.md (The "About Me" / Project Documentation)Ecommerce Logistics Optimization: Profit Recovery Analysis🎯 Project OverviewThis project identifies geographic and product-level "profit leakage" caused by outdated flat-rate shipping models. By analyzing a pilot dataset of e-commerce transactions, I developed a Volume-Based Tiered Shipping Model to recover margins and incentivize bulk purchasing.🛠️ Tools UsedSQL (PostgreSQL/SQLite): Data cleaning, relational mapping, and metric engineering.Tableau: Interactive dashboarding and "What-If" scenario modeling.Excel/CSV: Raw data source for sales and logistics costs.⚙️ The ProcessData Engineering: Standardized inconsistent geographic data (e.g., "IN." to "Indiana") and joined fragmented sales/shipping tables using SQL.Audit & Validation: Performed a Cross-Table Audit to ensure 100% relational integrity across regions.Scenario Modeling: Created calculated fields in Tableau to compare the Current State against a Hypothesis Model (using quantity-based multipliers from $1.0$ down to $0.3$).UI/UX Design: Built a stakeholder-ready dashboard with global filters for Category and State exploration.💡 Key InsightsIdentified Leakage: Discovered that heavy categories (Pet Food/Electronics) in long-distance states like Washington were creating a 65% shipping-to-revenue ratio.Product Impact: Isolated a single high-volume product causing a $5,000 annual loss due to unoptimized shipping.Profit Recovery: Demonstrated that a tiered shipping strategy increases total net profit from $250K to $411K—a 64% increase—without changing product prices.📊 Dashboard ViewGeographic Audit Map: Highlights "Red" loss zones in real-time.Margin-Eroding Products: A Top 5 bar chart identifying the biggest "profit killers."Strategy Comparison: A definitive 3-bar visual showing the financial impact of the proposed logistics shift.
